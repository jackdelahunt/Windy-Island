## Project overview
- A realtime 3D scene of a island landscape
- Runs in the browser using webgl2 for 3D graphics
- Built as a single html file with all JavaScript and assets embeded

## Structure
- Source code: `src/*.ts`
- Entry point: `src/main.ts`
- Models: `src/assets/models/`
- Textures: `src/assets/textures/`
- Shaders: `src/assets/shaders/`

## Building
- Do not attempt to run any build command it will fail, ask user to give feedback on changes instead
- Do not attempt to check your changes with git as it will fail
